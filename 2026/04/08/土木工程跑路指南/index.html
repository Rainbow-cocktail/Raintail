从零开始学CV-土木跑路指南01

土木工程学习计算机

前言：

在我写下这篇文章的时候，恰逢论文即将盲审，我其实会怀着十分忐忑的心情，因为毕竟论文送审结果出来之前，还是要时刻担心自己能否顺利毕业，流程是否一切完备，但我也觉得，如果现在不写下来，等以后解放的时候，估计已经忘光了，就趁着自己还是在校硕士生的身份，好好写这篇博客

我写的经验当然只是我看到的，我所做的，我所遇到的，一家之言，因此肯定有所疏忽，如果能带给此刻为自己专业和前景感到迷茫你的一点小小的帮助，就足够了

我接下来会从学习、科研和工作三个大方面展开

学习篇

对于想要入门计算机视觉的同学来说，现在已经非常的便捷了，都2026年了，我觉得我肯定没必要提到为什么计算机视觉很水很不好找工作等等话题，而且我也不想谈什么自学能力的问题，这里是我的经验分享而不是讨论学习的意义！

深度学习入门

下面是我推荐的计算机视觉入门的路线：

首先直接壁垒任何b站上搜到的卖课课程，点名“某迪哥”。

我建议先学习 Python，然后简单了解一些常用的 Python 附属库，包括但不限于 NumPy、Pandas、Matplotlib。这些都是了解即可，不用花大把的时间。

我推荐“鱼书”，这本书全程使用 NumPy，没有依赖任何较新的深度学习框架，而是从底层手写神经网络，内容通俗易懂，非常适合入门。

在课程方面，我推荐刘二大人的“深度学习入门课”。他是一名高校教师，这门本科生课程讲解简单清晰。跟着他完成相关 demo，即可对 MLP、CNN、RNN 等最基础的深度学习模型有一个整体认识。

前面这些我都只能称之为入门阶段（皮毛）。接下来是比较关键的部分，我这里先不讨论自学能力等因素，直接无脑推荐课程。

  • B站“小土堆”——深度学习入门课 + 实战课
  • “动手学深度学习”——李沐

这两个课程是构建深度学习基础非常扎实的选择，覆盖了环境配置、实操训练以及相关数学基础。内容方面，基本涵盖了从底层原理到具体任务的完整路径，包括分类任务、目标检测、图像相关任务（如风格迁移、GAN 等），以及序列预测任务、机器翻译任务等。同时也涉及 RNN、GRU、LSTM、Transformer 等主流模型架构。

当你看到李沐课程的最后一个视频时，我相信你已经基本走完了近些年的深度学习发展脉络，也已经对整个 AI 领域有了一个整体性的认识。

计算机视觉提升

在完成前面的课程后，我相信你已经对计算机视觉有了基本认知，而且之前的实操项目大多也与视觉相关。因此，这一阶段更多是补充知识和强化实践。

首先是 Ultralytics。这是目前计算机视觉领域中几乎无法绕开的工具库。你需要进入其官网，查阅官方文档，按照步骤逐步实操，运行示例代码。同时，可以通过合适的方式访问 YouTube，观看官方教学视频。

Ultralytics YOLO 是当前非常主流的检测框架之一，上手门槛较低，也比较容易理解。因此，想要弄清其整体架构，建议先从实际使用入手。

当你能够熟练使用之后，下一步就是深入理解其原理。我建议可以多看一些个人博客以及 B 站 up 主的视频，至少需要弄清楚目标检测的基本原理，以及 YOLOv1 ~ YOLOv3、YOLOv5、YOLOv8、YOLOv11 等经典检测模型。

由于这类视频资源非常多，这里就不做具体推荐了。

接下来我推荐B站的两个up

  • 霹雳吧啦Wz（霹雳吧啦Wz的个人空间-霹雳吧啦Wz个人主页-哔哩哔哩视频）
  • Bubbliiiing（Bubbliiiing的个人空间-Bubbliiiing个人主页-哔哩哔哩视频）

这两个 up 主基本是手把手教学，使用 PyTorch 从零搭建网络，很多内容都是“手搓”实现。

如果时间充裕，可以系统地看完；如果时间有限，其实看一两个视频之后，就可以直接去他们的仓库找源码，需要用到什么网络再针对性学习即可。

这里推荐重点关注以下三个模型：

  • U-Net
  • ViT
  • Swin Transformer

理论同时也要跟上，我推荐 沐神的读论文系列， 你可以边往多模态发展：

  • Transformer论文逐段精读【论文精读】
  • ViT论文逐段精读【论文精读】
  • Swin Transformer论文精读【论文精读】
  • CLIP 论文逐段精读【论文精读】

这些都是非常好的视频，理论和实践一定要跟上。

当然提升最好的方式是属于实践中学习，如果能有科研项目，或者自己的课题边做，那提升会非常快的，采用自己搭建的网络，跑自己的数据集，完成自己的课题是非常有成就的事情

此外，你可能已经注意到，Ultralytics 是一个围绕 YOLO 的视觉任务框架。那么，对于其他网络，是否也存在类似这种“傻瓜式”的框架，比如用于分割任务的呢？其实是有的，比如 MMSegmentation。

如果只是进行基础的 RGB 分割任务，这个框架是一个不错的选择，上手也比较方便。不过，由于它的拓展性相对较弱，而且与我的课题方向不太契合，所以我没有进行深入研究。

如果有相关需求，这个框架还是可以考虑使用的。

多模态图像融合

这是一个相对特殊的话题，与我的课题方向直接相关，因此单独放在这里。如果你也有相关需求，建议参阅以下论文与项目：

  • Comparison of multimodal RGB-thermal fusion techniques for exterior wall multi-defect detection

对应仓库：

OnionDoctor/mmUNet: Multi-modal UNet for Exterior Wall Multi-Defect Detection

此外，可以参考以下仓库及其相关论文：

  • wandahangFY/YOLOv11-RGBT: YOLOv11-RGBT: Towards a Comprehensive Single-Stage Multispectral Object Detection Framework（Supports RGBT detection for all YOLO series from YOLOv3 to YOLOv13, as well as RTDETR. 【Ultralytics YOLOv3-YOLOv13】 和有关论文

我额外提一嘴 wandahangFY，这个人的框架做得非常不错；同时作者在知乎上也有手把手的教程，教你如何改魔改 Ultralytics，实用性很强。
